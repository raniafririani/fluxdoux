# fluxdoux
FluxDoux adalah aplikasi berbasis web sederhana yang dirancang untuk membantu pengguna mengelola keuangan pribadi dengan tampilan yang elegan dan minimalis. Proyek ini dikembangkan dengan fokus pada pengalaman pengguna (User Experience) dan efisiensi penyimpanan data lokal

Fitur Utama
• Real-time Dashboard: Visualisasi arus kas (cash flow) menggunakan Chart.js untuk memantau saldo berjalan.

• Smart Input: Fitur otomatisasi format ribuan (titik) saat mengetik nominal saldo untuk mencegah kesalahan input.

• Transaction Management: Pencatatan pemasukan dan pengeluaran secara terpisah dengan kemampuan menghapus data secara satuan maupun keseluruhan.

• Responsive Design: Tampilan yang adaptif di berbagai perangkat (Mobile, Tablet, dan Desktop).

• Personalization: Fitur profil pengguna yang memungkinkan kustomisasi nama dan foto profil.
